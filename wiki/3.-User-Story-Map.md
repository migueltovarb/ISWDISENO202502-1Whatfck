# User Story Map

## Epic 1: Gestión de Usuarios

- US1: Como usuario, quiero registrarme con email y contraseña para crear una cuenta.
- US2: Como usuario, quiero iniciar sesión para acceder al sistema.

## Epic 2: Gestión de Reclamos (Cliente)

- US3: Como cliente, quiero registrar un nuevo reclamo con título, descripción, tipo de servicio y ubicación para reportar un problema.
- US4: Como cliente, quiero ver una lista de mis solicitudes activas en el dashboard al iniciar sesión.
- US5: Como cliente, quiero ver el detalle de un reclamo con un timeline de mensajes y respuestas.
- US6: Como cliente, quiero actualizar los detalles de mi reclamo si hay cambios.
- US7: Como cliente, quiero recibir notificaciones por email cuando el estado cambie.

## Epic 3: Gestión de Soporte

- US8: Como soporte, quiero ver lista de reclamos pendientes para asignarlos.
- US9: Como soporte, quiero responder a un reclamo con una solución propuesta.
- US10: Como soporte, quiero marcar un reclamo como resuelto para proponer su cierre.
- US11: Como soporte, quiero ver el detalle completo de un reclamo con el hilo de conversación.

## Epic 4: Gestión Administrativa

- US12: Como administrador, quiero cambiar el estado de cualquier reclamo.
- US13: Como administrador, quiero cerrar reclamos definitivamente.
- US14: Como administrador, quiero archivar un reclamo para ocultarlo del usuario sin eliminarlo de la base de datos.
- US15: Como administrador, quiero ver el detalle completo de cualquier reclamo.
